# Minesweeper (Java)

A console-based implementation of the classic Minesweeper game built using Java.

## Features
- Custom board size (user-defined rows and columns)
- Random bomb placement (no overlapping)
- Automatic neighbour bomb count calculation
- Flag and doubt system
- Score tracking
- Final bomb reveal on game over

## Concepts Used
- 2D Arrays
- Nested Loops
- Conditional Logic
- Random Number Generation
- Game State Management

## How to Run
1. Compile:
   javac minesweeper.java
2. Run:
   java minesweeper
